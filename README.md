# Group Attendance Automation

A Python-based desktop application designed to automate the processing of group attendance and specific dietary requirements. 

> **Note:** This is a showcase repository. The source code is kept private to ensure compliance with data protection standards, as the application processes sensitive personal information.

## 📌 Project Overview

Managing attendance and dietary restrictions manually for large groups is prone to human error. This tool streamlines the process by integrating directly with cloud data sources and providing a clean graphical interface for data management.

### Key Features
- **Data Integration:** Automated fetching and parsing of attendance records directly from Google Sheets.
- **Data Sorting:** Dynamic categorization of participants into specific groups while strictly filtering out invalid parsing artifacts.
- **Dietary Tracking:** Automated mapping of individual dietary requirements to specific participants.
- **User Interface:** Modern, responsive desktop GUI for intuitive data handling.

## 🛠️ Tech Stack

- **Language:** Python 3
- **GUI Framework:** CustomTkinter
- **Data Source:** Google Sheets API
